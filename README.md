# Alura-store-analysis
Data analysis project to evaluate the performance of Alura Store branches and recommend which store should be sold.

# 📊 Análisis de Desempeño de Tiendas – Alura Store

## 📌 Descripción del proyecto

Este proyecto tiene como objetivo analizar el desempeño de las cuatro tiendas de **Alura Store** para ayudar al **Sr. Juan** a tomar una decisión estratégica sobre **qué tienda vender para invertir en un nuevo negocio**.

Para ello se realizó un análisis exploratorio de datos evaluando distintos indicadores de desempeño, incluyendo ventas, satisfacción del cliente, categorías de productos y costos de envío.

A partir del análisis se generaron visualizaciones y un informe final que permite identificar cuál tienda presenta el menor rendimiento.

---

# 🎯 Objetivos del análisis

El análisis se enfocó en responder las siguientes preguntas:

- ¿Cuál tienda genera **mayores ingresos**?
- ¿Qué **categorías de productos** tienen mayor demanda?
- ¿Cuál es el **nivel de satisfacción de los clientes** en cada tienda?
- ¿Cuáles son los **productos más y menos vendidos**?
- ¿Cuál es el **costo promedio de envío** en cada tienda?

Estas métricas permiten comparar el desempeño de cada tienda y tomar una decisión informada.

---


El notebook contiene:

1. Carga y exploración de datos
2. Análisis de ventas
3. Análisis de categorías
4. Evaluación de satisfacción del cliente
5. Análisis de productos más y menos vendidos
6. Análisis de costos de envío
7. Visualizaciones con Matplotlib
8. Informe final
